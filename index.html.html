給你的行動建議
Action Guide
GUIDE
💡

手邊有 300 萬以上閒置現金，且不考慮提早發病風險
→ 現金信託可作為補充工具

🛡️

希望用最低預算、立刻啟動完整保障，對抗「錢還沒存夠就先倒下」的風險
→ 長照保險是目前最有效率的解方

⭐

最理想配置
→ 保險為主（建立基礎保障）
＋ 信託為輔（累積備用準備金）

⏰

保費通常隨年齡增長而遞增（實際依各險種費率表為準），核保限制愈多
→ 現在就是最好的時機
